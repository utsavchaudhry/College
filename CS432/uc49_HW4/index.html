Keyboard Controls

  • Q / W / E / A / S / D: Apply the selected transformation (Translate / Rotate / Scale) along the X, Y, or Z axis.
  • M: Increase the transformation delta by 0.001 (up to 0.030).
  • N: Decrease the transformation delta by 0.001 (down to 0.002).
  • R: Reset position → (0,0,0), rotation → (π/6, π/4, 0), scale → (1,1,1), delta → 0.01.
