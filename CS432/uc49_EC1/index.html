Interface Instructions (Updated):
• Animations start automatically and run seamlessly on load: squares rotate clockwise; the triangle rotates counter-clockwise; the circle “breathes” via smooth sinusoidal scaling.
• Use the on-screen controls (e.g., Rotate Ellipse button, Square Color dropdown) to modify graphics in real time.
• Press r, g, b, y, p, or w to set the Triangle color to red, green, blue, yellow, purple, or white, respectively.
• Left-click anywhere on the canvas to spawn a random regular polygon with N sides, where N ∈ [4, 19]; a new random N is chosen on each click.
Your browser does not support WebGL.
